  1. Python 为弱类型语言，区分大小写，注释用 # 号

  2. Python2默认不支持中文，需在程序开头添加注释#coding=utf-8或者-*- coding:utf-8 -*-(python语法规范推荐使用后者)。

  3. Python基本数据类型

    • 复数表示方法为 1+2j(不是i，是j)。
    • 布尔类型赋值只能用True 或False (必须首字母大写！) ，不能用 0 或 1。

  4. 格式化输出使用 % 号

    Python 中单双引号通用，嵌套使用可避免在定义字符串时转义字符的使用。

  5. Python2 ：

    • raw_input()会把用户输入的任何值都作为字符串来对待。
    • input()函数接受的输入必须是表达式。

    Python3：

    • 只有input(),作用等同于python2中的raw_input。

  8. 字符串常见操作，以下所有方法均不改变原字符串

    • mystr.find(“hello”, 0, 10)

      查找子字符串，返回开始的索引值，没有找到则返回-1，后两个参数可省。

      rfind() 用法相同，从右边开始查找

    • mystr.index(“hello”, 0, 10)

      同find()方法，找不到会报一个异常。ValueError: substring not found

      rindex() 用法相同，从右边开始查找

    • mystr.count(“hello”, 0, 10)

      返回子字符串出现的次数，后两个参数可省。

    • mystr.replace(str1, str2, count)

      把mystr中的 str1 替换成 str2 ,替换次数不超过 count 次，count可省，返回替换后的字符串，原字符串不变。

    • mystr.split(str, num)

      按 str 切分 mystr, str 为 None 或未指定时默认使用任意空白字符，切分次数不超过 num 次，num可省，返回切分后的字符串组成的list，原字符串不变。

    • mystr.capitalize()

      首字母大写，其余字母小写

    • mystr.title()

      每个单词首字母大写，其余字母小写

    • mystr.starswith(“hello”)

      检查字符串是否以 hello开头，返回True 或 False

    • mystr.endswith(“hello”)

      检查字符出是否以hello结束，返回True 或 False

    • mystr.lower()

      全部转换为小写

    • mystr.upper()

      全部转换为大写

    • mystr.ljust(width)

      返回一个原字符串左对齐，并使用空格填充至长度width 的新字符串

    • mystr.rjust(width)

      返回一个原字符串右对齐，并使用空格填充至长度width 的新字符串

    • mystr.center(width)

      返回一个原字符串居中，并使用空格填充至长度width 的新字符串

    • mystr.lstrip()

      删除 mystr 左边的空白字符

    • mystr.rstrip()

      删除 mystr 右边的空白字符

    • mystr.strip()

      删除 mystr 字符串两端的空白字符

    • mystr.partition(str)

      把 mystr 以 str 分割成三部分，str前，str 和 str后，返回一个元组

      rpartition() 用法相同，从右边开始

    • mystr.splitlines()

      按行分隔，返回一个包含各行作为元素的list

    • mystr.isalpha()

      判断 mystr 所有字符是否都是字符，返回 True 或 False

    • mystr.isdigit()

      判断 mystr 所有字符是否都是数字，返回 True 或 False

    • mystr.isalnum()

      判断 mystr 所有字符是否都是字母或数字，返回 True 或 False

    • mystr.isspace()

      判断 mystr 所有字符是否都是空白字符（空格，table，回车），返回 True 或 False

    • mystr.join(iterable)

      将 mystr 插入 iterable 的间隔，构造出一个新的字符串，iterable 为可迭代类型，可以是 list, tuple 或 str

  9. 列表中的元素可以是不同类型

    • append 向列表添加元素

      a.append(b)表示将b元素添加到a列表，即使b也是一个列表

    • extend 向列表中添加另一个列表中的元素

      a.extend(b)表示将b中的元素逐一添加到列表a中

    • insert(index, object) 在指定位置index前插入元素object

    • index 和 count 与字符串中的用法相同

    • del, pop, remove

      del:根据下标进行删除

      pop:删除最后一个元素

      remove: 根据元素的值进行删除

    • sort, reverse

      sort方法是将list按特定顺序重新排列，默认为由小到大，参数reverse=True可改为倒序，由大到小。

      reverse方法是将list逆置。

  16. 继承自object的类是新式类，不是继承object的类是经典类

  17. 以两个下划线开头两个下划线结尾的方法为魔法方法

    __init__(self)方法在创建一个对象时默认被调用，不需要手动调用

    即使没有显式定义__init__(self)方法，python也会自动隐式添加，所以所有的类都有该方法。

    • 在类内部获取 属性 和 实例方法，通过self获取；

    • 在类外部获取 属性 和 实例方法，通过对象名获取。

    • 如果一个类有多个对象，每个对象的属性是各自保存的，都有各自独立的地址；

    • 但是实例方法是所有对象共享的，只占用一份内存空间。类会通过self来判断是哪个对象调用了实例方法。

  18. 当使用print输出对象的时候，默认打印对象的内存地址。如果类定义了__str__(self)方法，那么就会打印从这个方法中 return 的数据，该方法有且只有一个self参数。__str__方法通常返回一个字符串，作为这个对象的描述信息。

  19. 当删除对象时，python解释器会默认调用__del__()方法

    当有变量保存了一个对象的引用时，此对象的引用计数就会加1，当使用del() 删除变量指向的对象时，则会减少对象的引用计数。如果对象的引用计数不为1，那么会让这个对象的引用计数减1，当对象的引用计数为0的时候，则对象才会被真正删除（内存被回收）。程序运行结束时会自动销毁所有对象。

  20. Python允许多继承，子类会继承所有父类的属性和方法。如果多个父类中有同名的 属性和方法，则默认使用第一个父类的属性和方法（子类的魔法属性__mro__决定了属性和方法的查找顺序）

  21. 在调用父类方法时，通过传递self参数，来控制方法和属性的访问修改。

    子类中调用父类方法

    如果继承了多个父类，且父类都有同名方法，则默认执行第一个父类的

  22. 属性名和方法名前加两个下划线表示私有，不能在外部通过对象名访问。

    Python中没有像C++中 public 和 private 这些关键字来区别公有属性和私有属性。

    Python是以属性命名方式来区分，如果在属性和方法名前面加了2个下划线’__’，则表明该属性和方法是私有权限，否则为公有权限。

  23. 如果需要在类外修改类属性，必须通过类对象去引用然后进行修改。如果通过实例对象去引用，会产生一个同名的实例属性，这种方式修改的是实例属性，不会影响到类属性，并且之后如果通过实例对象去引用该名称的属性，实例属性会强制屏蔽掉类属性，即引用的是实例属性，除非删除了该实例属性。

  24. 类方法使用@classmethod标识，第一个参数为该类对象，一般命名为cls。

    静态方法使用@staticmethod标识，不需要定义参数。

    • 类方法的第一个参数是类对象cls，那么通过cls引用的必定是类对象的属性和方法。

    • 实例方法的第一个参数是实例对象self，那么通过self引用的可能是类属性、也有可能是实例属性（这个需要具体分析），不过在存在相同名称的类属性和实例属性的情况下，实例属性优先级更高。

    • 静态方法中不需要额外定义参数，因此在静态方法中引用类属性的话，必须通过类对象来引用

  25. __new__和__init__方法

    • __new__至少要有一个参数cls，代表要实例化的类，此参数在实例化时由Python解释器自动提供

    • __new__必须要有返回值，返回实例化出来的实例，这点在自己实现__new__时要特别注意，可以return父类__new__出来的实例，或者直接是object的__new__出来的实例

    • __init__有一个参数self，就是这个__new__返回的实例，Python 解释器在拿到__new__返回的实例后将其传给__init__,__init__在__new__的基础上可以完成一些其它初始化的动作，__init__不需要返回值

    • 定义对象时传的参数__new__和__init__都能拿到，可以不用，但必须接收

    • 如果没有捕获到异常，那么就执行 else 中的语句

    • 关于代码super().__init__()的说明

      因为__init__方法往往是用来对创建完的对象进行初始化工作，如果在子类中重写了父类的__init__方法，即意味着父类中的很多初始化工作没有做，这样就不保证程序的稳定了，所以如果重写了父类的__init__方法，最好是先调用父类的这个方法，然后再添加自己的功能

  1. 将某个模块以整体形式导入，以模块名.函数名 的形式调用

    导入某个模块的某个或某几个函数，直接以函数名调用

    不仅可以引入函数，还可以引入一些全局变量、类等。通过这种方式引入的时候，调用函数时只能给出函数名，不能给出模块名，当两个模块中含有相同名称函数的时候，后面一次引入会覆盖前一次引入。也就是说假如模块A中有函数function( )，在模块B中也有函数function( )，如果引入A中的function在先、B中的function在后，那么当调用function函数的时候，会执行模块B中的function函数。

    导入一个模块中所有内容

    若导入时使用as 指定别名，则只可使用别名调用

    导入模块时 Python解释器对模块的搜索顺序是：

    ​ 1.当前目录 –> 2.PYTHONPATH变量中指定的目录 –> 3.默认路径（Python安装路径）

    system 模块的 sys.path 变量的内容即为模块搜索路径，包含 当前目录，PYTHONPATH指定目录和默认目录。

  2. __name__变量用于指示所属文件是作为主程序使用还是作为模块被使用，当作为主程序使用时，__name__被赋值为__main__,当作为模块被使用时，__name__被赋值为模块名。常用来有选择的执行测试代码。

  3. 在以 from xxx import * 的形式导入模块所有内容时，模块中__all__变量指定的元素才会被导入，不写__all__变量则默认全部导入，以其他形式导入模块不受__all__变量的限制。__all__变量是一个元素为字符串的列表。

    • 以 from xxx import * 的形式导入包中的所有模块时，包中必须有__init__.py文件，且只有其中的__all__变量指定的模块会被导入，__init__.py中的import语句中的包不会导入到当前文件，但在执行__init__.py中的语句时是有效的，__init__.py中的语句在导入时会被执行，可用其进行初始化工作。

    • 以包的形式导入时，Python2 要求目录下必须有__init__.py文件，Python3中具体到某一模块时可以没有__init__.py文件，其余情况也要求有__init__.py文件。要导入的模块必须在__init__.py中以 import 语句写出，Python2 中可以写 import xxx或from . import xxx,Python3只支持后一种格式。

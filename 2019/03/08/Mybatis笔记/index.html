Tips

  1. Mybatis-config.xml有严格的顺序

    • configuration 配置
      • properties 属性
      • settings 设置
      • typeAliases 类型别名
      • typeHandlers 类型处理器
      • objectFactory 对象工厂
      • plugins 插件
      • environments 环境
        • environment 环境变量
          • transactionManager 事务管理器
          • dataSource 数据源
      • databaseIdProvider 数据库厂商标识
      • mappers 映射器

    ​

    包装类型将首字母小写即可,BigDecimal的别名为bigdecimal或decimal
